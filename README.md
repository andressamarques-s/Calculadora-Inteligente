# Calculadora Inteligente

Projeto simples desenvolvido em Python para praticar:

- Funções
- Estruturas condicionais
- Laços de repetição
- Tratamento de erros
- Histórico de operações

## Funcionalidades

Soma  
Subtração  
Multiplicação  
Divisão  
Histórico  
Encerrar programa digitando "sair"
